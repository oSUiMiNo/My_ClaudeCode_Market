# 指示
- `$ARGUMENTS` 配下の全ディレクトリ階層構造を把握し報告せよ。
- 各ディレクトリに対し、中の様子を調査する
`出力形式`のような形で様子を記載せよ

# 手順
1. `README.md` があれば最初に読む
2. `$ARGUMENTS` 配下の全ディレクトリ構造を把握し、各ディレクトリ１つずつに対して /analyze_dir.md [ディレクトリの絶対パス] を実行する
3. analyze_dir から各ディレクトリの概要を受け取りる
3. 下記`報告様式`通りに情報をまとめる
